# StackWave - The Operating System for Tokenized Economies

StackWave is an AI-powered Web3 governance and analytics platform designed for the next generation of tokenized organizations.

## Project Structure
- `client/`: React frontend with Next.js/Tailwind
- `server/`: Express backend with PostgreSQL/Drizzle
- `contracts/`: Solidity smart contracts (Token, Staking, Payments)
- `shared/`: Shared TypeScript types and API routes
- `docs/`: Whitepaper, Pitch Deck, Tokenomics, and Security Audit

## Getting Started
1. Install dependencies: `npm install`
2. Run migrations: `npm run db:push`
3. Start development server: `npm run dev`

## Features
- **Governance:** Proposal creation and token-weighted voting.
- **Staking:** SWAVE token staking with yield tracking.
- **AI Analytics:** Wallet intelligence and risk assessment.
- **Web3 Auth:** Secure Sign-In With Ethereum (SIWE).
